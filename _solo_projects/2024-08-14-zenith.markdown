<h3>Technical Implementation</h3>
<ul>
  <li><strong>Odin Programming:</strong> Developed the entire game using the Odin programming language for optimal performance and modern language features</li>

  <li><strong>Puzzle Systems:</strong> Created challenging puzzle mechanics that integrate seamlessly with platforming and exploration</li>
  <li><strong>Game Architecture:</strong> Designed modular game systems for entity management, level loading, and state management</li>
